We are a group of developers who enjoy making Free and Open Source software.

Code

We share our code in our Github page. Here's a few projects we've worked on:

  • Kuvia, a simple image gallery for static web sites. Give it a list of image files, and you've got an image gallery that can be hosted in any web host.
  • Yet Another Hints Extension, an extension for Chrome that allows users to activate links without touching the mouse.
  • Kiteytys, a reporting backend for the Topaasia game.
